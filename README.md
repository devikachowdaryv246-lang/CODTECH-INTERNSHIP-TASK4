# CodTech Internship Task 4: Sentiment Analysis

## Project Description
This project implements a Natural Language Processing (NLP) model to perform sentiment analysis on textual data. It classifies text into Positive, Negative, or Neutral categories.

## Features
- **Data Preprocessing:** Cleaning text by removing special characters and noise.
- **Sentiment Scoring:** Using TextBlob to calculate polarity scores.
- **Insights:** Providing a summary of sentiment distribution.

## How to Run
1. Clone the repo: `git clone <your-link>`
2. Install dependencies: `pip install pandas textblob`
3. Run the notebook: `sentiment_analysis.ipynb`
